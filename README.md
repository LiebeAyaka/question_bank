# 西班牙语题库系统

**版本**: 0.3.1 |
**AI 协作**：本项目由 AI 辅助开发

题库管理系统，支持题目管理、筛选和试卷生成。

## 技术栈

| 层级 | 技术 |
|------|------|
| 前端框架 | React 18.2.0 |
| 语言 | TypeScript 5.3 |
| 构建工具 | Vite 5.0 |
| 状态管理 | Zustand 4.5 |
| 后端框架 | Flask 3.0.0 |
| 数据库 | SQLite |
| 数据验证 | Pydantic 2.5 |

## 快速开始

### 一键启动

**Windows**: 双击运行 `start.bat`

**Linux/macOS**: 运行 `start.sh`（需要先执行 `chmod +x start.sh` 赋予执行权限）

一键脚本会自动：
- 检查 Node.js 和 Python 环境
- 安装前端依赖（如需要）
- 创建 Python 虚拟环境并安装依赖（如需要）
- 启动前后端服务

启动后访问 http://localhost:10289

### 手动启动

#### 1. 启动后端

```bash
cd server
pip3 install -r requirements.txt
python3 app.py
```

后端运行在 http://localhost:10288

### 2. 启动前端

```bash
npm install
npm run dev
```

前端运行在 http://localhost:10289

### 3. 访问系统

打开浏览器访问 http://localhost:10289

### 4. 首次使用

首次启动时，如果数据库为空，系统会显示"启用示例题目"按钮。点击后会自动导入示例数据。

## 功能

- 题目管理（创建、编辑、删除）
- 按题型筛选题目
- 按难度筛选题目（简单/中等/困难）
- 按单元筛选题目
- 按考点模糊筛选
- 搜索题目内容
- 为题目标记**难度**（简单 / 中等 / 困难）
- 为题目标记**单元**
- 管理题目**考点**
- **出题策略管理**：创建/编辑/删除智能出题策略模板（集成在试卷生成页面）
- 生成试卷（支持 8 种题型）
- **智能出题**：根据策略自动按难度/单元比例分配题目
- 导出试卷为 Markdown 格式

## 支持题型

- 单选题 (single)
- 判断题 (judge)
- 阅读理解 (reading)
- 完形填空 (cloze)
- 任务型阅读 (task_reading)
- 填空题 (fill)
- 问答题 (short_answer)
- 作文题 (essay)
- **听力单选题 (listening_single)** - 一段材料配一道单选题
- **听力理解 (listening_group)** - 一段材料配多道子题（单选/填空混合）

## 环境变量

### 后端 (server/)

| 变量 | 默认值 | 说明 |
|------|--------|------|
| FLASK_DEBUG | false | 调试模式，生产环境勿设为 true |
| CORS_ORIGINS | http://localhost:10289 | 允许的跨域来源，多个用逗号分隔 |

### 前端

前端通过 Vite 代理转发 /api 请求到后端，无需配置 API 地址。
